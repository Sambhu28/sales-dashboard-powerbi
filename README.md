# Sales Dashboard — Power BI + Python

## Project Overview
Interactive sales dashboard analyzing retail store 
performance across USA built using Python and Power BI.

## Tools Used
- Python (Pandas) — Data Cleaning
- Power BI — Dashboard and Visualization

## Dashboard Pages
1. Sales Overview — KPIs, Monthly Trend, Category Sales
2. Regional Performance — Map, Profit by Region
3. Product Analysis — Top 10 Products, Profit Margin

## Key Insights
- Technology is the highest selling category
- West region generates the most profit
- Total sales of 2.33M across all categories
